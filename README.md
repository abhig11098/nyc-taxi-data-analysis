# nyc-taxi-data-analysis
Exploratory Data Analysis of NYC Taxi Trip Dataset using Python, Pandas, NumPy and Matplotlib.

This project performs Exploratory Data Analysis (EDA) on the NYC Taxi Trip dataset using Python.

- Cleaned and preprocessed large-scale taxi trip data using Pandas and NumPy.
- Performed missing value treatment and outlier detection.
- Analyzed trip distance, fare amount, passenger count, and travel patterns.
- Identified peak demand hours and ride distribution trends.
- Conducted statistical analysis to uncover business insights.
- Created visualizations using Matplotlib to support decision-making.
- Evaluated relationships between trip distance and fare amount.
- Generated actionable recommendations based on customer travel behavior.
- Built as part of a Data Science and Analytics learning initiative.

- Statistical Analysis
